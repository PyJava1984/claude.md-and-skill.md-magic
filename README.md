# AI Coding Standards Repository

A comprehensive collection of **`CLAUDE.md`**, **`SKILL.md`**, and technology-specific AI instruction files designed to help AI coding assistants produce consistent, maintainable, secure, and production-ready software.

Whether you're using Claude, ChatGPT, Gemini, or another AI assistant, this repository provides reusable standards, engineering guidelines, and domain knowledge that improve code quality and AI-assisted development workflows.

## Deliverables

### 1. `CLAUDE.md` — AI Coding Constitution

A comprehensive instruction file that defines how AI should behave while working inside a software repository.

Topics include:

* AI assistant behavior
* Repository awareness
* Incremental development
* Refactoring policy
* Test-first mindset
* Documentation requirements
* Security defaults
* Performance defaults
* Error handling
* Logging standards
* Code review checklist
* Architecture rules
* SOLID principles

* Twelve-Factor App principles

* DevOps considerations
* Git workflow

* Forbidden coding patterns

* AI self-review checklist
* AI verification checklist

## Goal

The objective of this repository is to provide a reusable foundation for AI-assisted software engineering by combining:

* Repository-level coding standards
* Technology-specific expertise
* Modern architecture guidelines
* Security-first development
* Performance best practices
* Production-ready engineering principles

Use these documents to ensure AI consistently produces clean, maintainable, scalable, and enterprise-grade software.
